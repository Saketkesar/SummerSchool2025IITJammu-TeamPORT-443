## 📋 Usage Instructions

1. **Launch:** `./run_shadowsnare.sh`
2. **Root Check:** Application will request privileges when needed
3. **Interface Selection:** Choose network interface from dropdown
4. **Network Scanning:** Auto-detect or manual range configuration
5. **Target Selection:** Choose targets from scan results
6. **Attack Launch:** Configure and execute MITM attacks
7. **Monitoring:** View real-time logs and captured credentials
